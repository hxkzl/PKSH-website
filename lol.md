北港高中學生會 (PHSA) 數位平台

專案概述

這個專案是國立北港高中學生會的官方數位平台，旨在提供一個現代、響應式（RWD）且功能完整的單一入口，以提升學生自治的效率與透明度。

核心功能 (MVP)

數位佈告欄： 提供最新公告與分類篩選。

匿名申訴信箱： 學生可匿名提交建議或申訴，並獲得追蹤 ID 查詢進度。

社團專區： 提供社團資訊瀏覽與分類篩選。

幹部後台： 供學生會幹部登入模擬介面，處理申訴案件與發布公告。

技術棧

前端： 單一 index.html 檔案 (HTML5, Vanilla JavaScript)

樣式： Tailwind CSS (透過 CDN 引入)

圖標： Lucide Icons (透過 CDN 引入)

部署： GitHub Pages
